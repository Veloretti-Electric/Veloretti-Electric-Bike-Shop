# Veloretti-Electric-Bike-Shop
Le projet Veloretti est une conception UX/UI d’un site web e-commerce pour la vente de vélos
électriques modernes. L’objectif principal est de créer une expérience utilisateur fluide et
élégante mettant en avant la qualité, le design et la performance des produits.
Le site propose une interface épurée avec une navigation intuitive, des pages produits détaillées
et un fort accent sur le visuel. Chaque section du site est pensée pour guider l’utilisateur vers
l’achat tout en valorisant l’esthétique du produit.
Le projet vise à offrir une expérience utilisateur moderne et responsive, adaptée à tous les
écrans.
